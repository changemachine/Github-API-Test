# Github API Test
## by James Nielson
### Date: Feb 19, 2016
#### Retrieves commits made by the entered Github user, via the Github API.  

#### Setup instructions
1. Clone git repo
2. $ npm install
3. $ gulp build
4. $ gulp serve
5. Project should load in your browser at localhost:3000
